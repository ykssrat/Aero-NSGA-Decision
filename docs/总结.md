# 航空电子设备元器件组合优化选型系统 (AeroOptiSelect)

## 工程命名候选方案

基于**“航空电子设备的元器件组合选型的最优化方法”**这一核心主题，这里提供几个不同侧重点的工程命名供您参考和后续替换：

### 1. 强调“航空/航天”与“优选” (推荐)
- **AeroOptiSelect** (Aeronautics + Optimization + Selection) —— 最凝练，具备国际化开源软件的命名风格。
- **AeroSysSelector** (Aero System Selector) —— 侧重于系统级的整体选型。
- **AvionicsOptima** (Avionics + Optima) —— 聚焦“航电系统”与“最优点”。

### 2. 强调“多目标”与“遗传算法”的学术风格
- **Aero-NSGA-Decision** (Aeronautics NSGA-II Decision System) —— 直白体现了底层使用的核心算法。
- **MOCAS-Avionics** (Multi-Objective Component Allocation System for Avionics) —— 标准的学术论文级命名（多目标航电组件分配系统）。
- **Avio-Pareto-Selector** —— 突出了寻找“帕累托前沿”这一核心数学机制。

### 3. 强调“复杂约束”与“高可靠性”
- **AeroConstraintOpt** (Aeronautics Constraint Optimizer) —— 强调解决复杂飞行限制条件。
- **Avionics-Reliant-Selector** —— 侧重于军工/航空领域对属性约束（如抗电磁干扰、飞行经历）的严苛筛选。

*(由于前面的文档中我们使用了 OptiSelect 作为占位，这里正式将本工程的文档名升级为 **AeroOptiSelect**)*

---

## 工程概述

**AeroOptiSelect** 是一套基于 **MATLAB** 编写的航电多目标优化与智能决策专家系统。它将 **NSGA-II（带精英保留策略的非支配排序遗传算法）** 与 **TOPSIS（逼近理想解排序法）** 两大经典算法级联，专为解决航空航天电子设备在严苛约束条件限制下的多组件/参数最佳配置选型问题而设计。

---

## 适用场景

- 航空航天电子设备的元器件组合选型
- 军工或雷达特种装备系统的多部件配置优化
- 任何需要在"成本"与"性能"之间做多目标权衡且附带强制属性约束（如国产化要求、抗电磁干扰要求、飞行经历认证要求等）的高可靠性决策场景

---

## 系统工作流程

```
用户交互输入航电筛选条件
        │
        ▼
  数据库预筛选（剔除不合格航电组件）
        │
        ▼
  NSGA-II 多目标遗传优化
  （100代进化，种群100）
        │
        ▼
  提取 Pareto 前沿解集
        │
        ▼
  TOPSIS 综合决策（权重加权）
        │
        ▼
  输出最优选型配置 + 可视化 + 方案落盘
```

## 核心技术要点

| 技术模块 | 说明 |
|---------|------|
| **NSGA-II** | 同时优化"最小化系统造价"与"最大化航电性能"两个互相矛盾的目标 |
| **TOPSIS** | 从 Pareto 前沿解集中挑选加权综合最优解 |
| **约束预筛选** | 支持属性3（如抗电磁干扰）、属性4（国产化率要求）、属性5（飞行经历/适航认证）三类布尔约束的组合筛选 |
| **国产化性能折扣** | 通过 `CalculateDomesticBoost` 模拟大量采用纯国产替代部件时，系统整体性能可能面临的非线性波动评估 |
| **整数编码** | 决策变量为离散硬件型号索引，采用整数交叉和整数变异算子防越界 |
